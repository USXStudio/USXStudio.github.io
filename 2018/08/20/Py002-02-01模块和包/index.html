模块

一个python文件就是一个模块

随着程序代码越来越多，在一个文件里代码越来越多，越来越难以维护。于是就把代码进行拆分。每个功能进行独立。–模块的诞生

这样，每个文件包含的代码就变少了。维护成本就低了

是不是很像树的结构那样。如某个功能修改仅仅需要修改对应节点的代码。

模块的好处

  • 提高可维护性
  • 可重用
  • 避免函数名和变量名冲突

模块分类

  • 内置模块，执行help(‘modules’) 查看所有python自带模块
  • 第三方模块，通过pip install 的模块
  • 自定义模块
